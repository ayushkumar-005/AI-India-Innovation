# AI India Innovation

> **Infrastructure First. Intelligence Second.**

A production-grade web platform designed for the fictional AI infrastructure company. This application facilitates IT Services, Document Digitization, Rental Services and Server Procurement, Data Center Setup and AI-Based Software Development

**Live Demo:** https://aiindiainnovation.vercel.app/

![Project Banner](https://i.ibb.co/5gzvy9Hq/chrome-TXLudu-Qap-H.png)

## Tech Stack

-   **Backend:** Node.js, Express.js (ES Modules)
-   **Frontend:** EJS Templating Engine, Tailwind CSS v4
-   **Security & Ops:** Helmet.js, Compression, Rate Limiting
-   **Deployment:** Vercel (Serverless adaptation)
